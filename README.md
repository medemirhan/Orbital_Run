# Orbital Run

Orbital Run game written in C++.

* Orbitron tries to survive by avoiding hitting the monsters and bombs.
* User can direct Orbitron to inner or outer orbits by pressing arrow keys.
* Entity velocities increase at each lap completion.
* Orbitron can collect Green Hearts. Once collected the required number of Green Hearts specified in the config file, Orbitron gets one more life.
* Orbitron can also collect Rocket Rights. User can fire the rockets in order to kill Monsters by pressing space bar.
* Monsters pursue Orbitron by changing orbits.
* Several options about the game are configurable by editing config.xml file.
